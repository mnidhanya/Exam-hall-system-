🏫 Exam Hall Management System
A frontend-based Exam Hall Management System built using HTML, CSS, and JavaScript.
This project was developed as part of my web development learning journey to understand core frontend concepts and DOM manipulation.
🚀 Features
Login screen UI
Exam hall switching (Hall A / Hall B)
Interactive seating layout
Click-based attendance marking (Free / Present / Absent)
Live attendance counter
Attendance table view
Export attendance as CSV file
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📌 How It Works
Seats are generated dynamically using JavaScript
Clicking a seat changes its attendance state
Attendance statistics update in real-time
Data can be exported as a CSV file for reporting
🎯 Purpose
This project was created to practice frontend development fundamentals including layout design, DOM manipulation, event handling, and interactive UI building.
